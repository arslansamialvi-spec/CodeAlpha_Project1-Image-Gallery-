# Image Gallery

A responsive and interactive Image Gallery built using HTML, CSS, and JavaScript. The project allows users to filter images by category and view them in an enlarged slideshow mode.

## Features

* View all images in a single gallery.
* Filter images by category:

  * Nature
  * City
* "All" button displays every image in the gallery.
* Click any image to open it in a larger view.
* Navigate through images using Previous (<) and Next (>) buttons.
* Close the enlarged image view using the Close button.
* Responsive design that works on both desktop and mobile devices.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

1. Open the application in a web browser.
2. Use the category buttons to filter images:

   * **All**: Displays all images.
   * **Nature**: Displays only nature-related images.
   * **City**: Displays only city-related images.
3. Click on any image to view it in a larger size.
4. Use the **Previous (<)** and **Next (>)** buttons to browse through images.
5. Click the **Close (X)** button to exit the enlarged image view.

## Project Structure

* `index.html` – Main webpage structure
* `styles.css` – Styling and responsive design
* `script.js` – Filtering functionality and image viewer logic
